# 概要
AutoHotkey v2.0の設定をPC間で共有するためのリポジトリ

## global_hotkeys.ahk ショートカット一覧

### 操作用ホットキー
| ショートカット | 動作 |
| --- | --- |
| `F18 + n` | 前のタブへ移動（Ctrl+Shift+Tab） |
| `F18 + m` | 次のタブへ移動（Ctrl+Tab） |
| `F19`（単押しのみ） | IME をオン/オフ切り替え |
| `F19 + n` | 音量を下げる |
| `F19 + m` | 音量を上げる |
| `F19 + ,` | メディアの再生/一時停止 |
| `F19 + p` | Ctrl+Shift+P を送信（コマンドパレット呼び出しなど） |
| `F19 + u` | 前のウィンドウへ移動（Alt+Shift+Tab） |
| `F19 + i` | 次のウィンドウへ移動（Alt+Tab） |
| `F19 + k` | PageUp（Shift押下中は Shift+PageUp） |
| `F19 + j` | PageDown（Shift押下中は Shift+PageDown） |
| `F19 + h` | Home（Shift押下中は Shift+Home） |
| `F19 + l` | End（Shift押下中は Shift+End） |
| `F17` | IMEがオフなら直前の英数字を消してIMEオンで再入力、オンならF10→Enterで英数確定してIMEオフ |
| `Ctrl + Space` | Claude を起動/表示/最小化をトグル（常に最前面化） |
| `F19 + 1` | 本日の日付 `YYYY年M月D日(曜)` を入力 |
| `F19 + 2` | 現在時刻 `HH:mm` を入力 |

備考: F19は単押し時のみIMEトグルが動作し、他のキーとの同時押し時はコンビネーションの動作のみ行われます。
